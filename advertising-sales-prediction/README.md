# Advertising Sales Prediction

Machine Learning project using Linear Regression to predict product sales based on TV, Radio, and Newspaper advertising budgets.

---

##  Project Overview

This project analyzes advertising data and builds a Linear Regression model to predict sales based on different advertising platforms.

The project includes:
- Data preprocessing
- Statistical analysis
- Correlation analysis
- Model training
- Prediction
- Performance evaluation
- Data visualization

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
